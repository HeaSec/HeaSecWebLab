# XXE基础靶场

## 靶场简介

欢迎来到天积数据平台！本靶场模拟了一个商品数据导入系统，系统支持通过XML文件批量导入商品数据。然而，系统在处理XML时存在安全配置不当的问题，导致存在XXE（XML外部实体注入）漏洞。

## 学习目标

- 理解XXE漏洞的基本原理，掌握不同场景下XXE漏洞的利用方式。

## 第三关提示

第三关需要自行搭建外部服务接收带外数据，可参考以下工具：

- webhook.site — 在线webhook接收服务
- Burp Collaborator — Burp Suite自带的外带服务
- 自建HTTP服务 — 在本地或其他服务器启动HTTP服务接收请求

## 重置方式

点击页面右上角的重置按钮即可重置靶场数据。
