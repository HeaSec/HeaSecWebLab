# SOAP与XML安全漏洞靶场

## 靶场简介

本靶场模拟"天积云服务"Web应用平台，展示SOAP Web服务中与XML相关的安全漏洞。平台前端与后端之间使用SOAP XML协议通信，由于后端在处理XML数据时存在多种安全漏洞，攻击者可以通过构造恶意XML内容实施攻击。

## 学习目标

- 理解SOAP服务中XML参数注入的原理和利用方式
- 掌握XPath注入攻击技巧
- 学习XXE漏洞如何被利用进行SSRF攻击
- 理解为什么前端安全过滤不能替代后端安全防护

## 任务要求

根据各关页面中的任务提示，利用SOAP请求中的XML安全漏洞获取通关密码。

## 攻击方式提示

- 建议使用Burp Suite等抓包工具拦截和修改HTTP请求
- 注意观察请求体是SOAP XML格式，可以修改XML内容
- 某些关卡可能需要通过抓包才能完成攻击

## 重置说明

页面右上角重置按钮可重置靶场数据，重置将清空所有通关密码和用户数据并恢复初始状态。
