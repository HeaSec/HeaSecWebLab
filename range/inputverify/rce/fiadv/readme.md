# 文件包含进阶 靶场

## 靶场说明

本靶场模拟"天积资源管理器"动态模板加载系统，系统存在文件包含漏洞且 allow_url_include 已开启（非默认PHP配置，需要手动开启）。在基础版文件包含靶场（LFI 信息读取）的基础上，进阶版聚焦于文件包含漏洞导致的**远程代码执行（RFI/RCE）**——攻击者可以通过多种协议让 include() 执行任意 PHP 代码。

## 学习目标

深入理解 PHP 文件包含漏洞的多协议利用方式

## 任务要求

利用文件包含漏洞通过各种协议执行 PHP 代码输出目标字符串 `I_love_heasec_xxxxxxxxxxxxxxxxxxxx`。每种不同的协议成功执行一次即可解锁一条成就记录，收集足够多的协议成就解锁星级奖励（1星/2星/3星）。

## 操作提示

- 你需要先获取完整的目标字符串（页面只显示前 8 位）。系统的模板配置目录中存放着一些配置文件，某些 PHP 伪协议可以帮助你在不执行代码的情况下查看文件的原始内容
- 获取目标字符串后，尝试使用不同的协议来执行代码。系统提供了文件上传功能，仔细思考上传的功能和模板加载的功能之间是否存在可以结合利用的地方
- 注意观察页面上方的 PHP 配置检测提示，如果allow_url_include未开启，某些协议可能无法正常工作。
