# 回显型命令注入 靶场

## 靶场说明

本靶场模拟"天积网络诊断工具"在线平台，提供IP地址连通性检测（ping）功能。系统在将用户输入拼接到系统命令时存在命令注入漏洞，且各关卡的安全过滤策略逐步增强。通过三关渐进式实操，帮助你掌握PHP回显型命令注入漏洞的基础利用技巧和过滤绕过方法。

## 学习目标

- 理解命令注入漏洞的基本原理
- 掌握常见命令分隔符的使用
- 理解黑名单过滤的固有缺陷，学会寻找过滤遗漏，掌握高级绕过技巧

## 任务要求

本靶场共三关，每关独立页面。在网络诊断工具的IP输入框中构造特殊输入，使系统执行目标命令并回显结果。

## 操作提示

- 每关成功执行目标命令后，系统会自动检测并显示通关状态
- 命令执行结果会在终端风格的输出区域中展示

## 重置方式

点击页面右上角的"重置"按钮，确认后将清空所有关卡通关状态，恢复初始状态。
