# 文件包含基础靶场

## 靶场简介

本靶场模拟一个名为**"天积文档查看器"**的在线文档浏览系统。系统支持用户通过 URL 参数选择要查看的页面文档，服务端使用 PHP 的 `include` 函数动态加载对应的页面文件并展示其内容。系统在处理用户提交的页面参数时存在文件包含漏洞，且各关卡的安全防护措施逐步增强。

## 学习目标

掌握 PHP 文件包含漏洞的基础利用技巧

## 任务要求

根据各关页面中的任务提示，构造合适的 page 参数（或结合上传功能）利用文件包含漏洞获取通关密码。可通过页面上的导航链接、路径输入框与文档查看器交互。

## 重置方式

点击页面右上角的**重置**按钮可重置靶场数据（清空所有通关密码和上传文件）。
