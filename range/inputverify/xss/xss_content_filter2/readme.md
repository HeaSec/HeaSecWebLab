# 标签与事件

## 靶场说明

学习在部分标签与事件被禁用的情况下，如何利用HTML标签与事件处理器的多种组合方式绕过黑名单限制，并熟悉各种各样的标签与事件处理器。

## 目标

探索并使用未被禁用的其他HTML标签以及事件处理器组合，成功触发 alert(heasec) 弹窗。

- 第一颗星：成功使用任意组合触发1次弹窗。
- 第二颗星：累计收集使用5个不同的事件类型或HTML标签成功触发弹窗。
- 第三颗星：累计收集使用10个不同的事件类型或HTML标签成功触发弹窗。

## 重置说明

靶场数据可以通过页面右上角的重置按钮进行重置。
重置将清空所有成就记录并恢复初始状态。
