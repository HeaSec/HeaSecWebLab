# 反序列化实战靶场

## 靶场简介

本靶场模拟"天积插件管理系统"CMS插件加载机制。系统支持通过上传序列化的插件配置数据来动态加载和执行插件功能，但在处理用户提交的插件配置数据时存在反序列化漏洞，且未做充分的安全处理。

## 学习目标

通过本靶场的实操练习，掌握PHP反序列化漏洞的高级利用技巧

## 任务要求

根据各关页面中的任务提示，构造恶意序列化字符串提交到对应的数据处理接口，利用漏洞机制获取通关密码。可点击"查看源代码"按钮了解当前关口的类定义和数据处理逻辑。

## 重置方式

点击页面右上角的重置按钮即可重置靶场数据，所有通关密码将被重新生成。
