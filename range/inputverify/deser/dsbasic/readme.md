# 反序列化基础 靶场

## 靶场说明

本靶场帮助你学习PHP反序列化漏洞的基本原理和利用方式。

通过8个章节的学习，你将掌握：

- 序列化与反序列化的基本概念
- PHP序列化格式详解
- 交互式序列化/反序列化实践
- PHP魔术方法与反序列化漏洞
- 反序列化漏洞利用流程
- POP链构造练习
- 防御措施

## 学习目标

1. 理解序列化与反序列化的原理和应用场景
2. 掌握PHP序列化字符串的格式
3. 理解魔术方法在反序列化中的触发机制
4. 学会构造POP链利用反序列化漏洞
5. 了解反序列化漏洞的防御方法

## 提示

- 序列化格式中的类型标识符：s(字符串)、i(整数)、a(数组)、O(对象)
- 重点关注 `__destruct()` 和 `__wakeup()` 魔术方法
- POP链的核心是：通过属性值串联多个类的魔术方法
