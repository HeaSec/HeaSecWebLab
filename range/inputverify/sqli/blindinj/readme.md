# SQL盲注靶场

## 靶场说明

本靶场模拟"天积商城后台管理系统"，系统存在SQL注入漏洞但不回显查询数据。你需要使用不同的盲注技术来获取各关卡的通关密码。

## 学习目标

通过本靶场，你将掌握SQL盲注的三种核心技术：

- **报错注入**：利用数据库报错函数提取数据
- **布尔盲注**：利用页面响应差异逐字符提取数据
- **时间盲注**：利用响应时间差异逐字符提取数据

## 任务要求

根据各关页面中的任务提示，利用不同的盲注技术获取通关密码。

## 重置方式

点击页面右上角的重置按钮可重置靶场数据，恢复初始状态。
