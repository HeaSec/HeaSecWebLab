# SQL关键字过滤靶场

## 靶场简介

天积商城运营管理系统存在SQL注入漏洞，系统部署了不同的关键字检测/过滤机制。你需要绕过关键字过滤，利用UNION查询获取每关的通关密码。

## 学习目标

掌握SQL注入中关键字过滤的常见绕过技巧。

## 任务要求

根据各关页面中的任务提示，绕过关键字过滤，利用UNION查询获取通关密码。

### 第一关：商品库存查询

系统会移除输入中的SQL关键字，但移除方式可能存在绕过空间。通关密码存储在 heasec_kwbpsi_goods 表中。

### 第二关：订单物流查询

系统升级了关键字检测机制，大小写混写不再有效。关键字检测基于字符串匹配，但SQL解析器如何理解关键字？通关密码存储在 heasec_kwbpsi_orders 表中。

### 第三关：客户反馈查询

系统部署了多重过滤器，除了关键字严格过滤外，还有空白字符和注释符的限制。通关密码存储在 heasec_kwbpsi_feedback 表中。

## 重置方式

页面右上角重置按钮
