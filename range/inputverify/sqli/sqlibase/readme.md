# SQL注入基础靶场

## 靶场介绍

天积科技资讯平台是一个模拟的企业内部信息发布系统，提供资讯浏览、搜索、用户登录、意见反馈、偏好设置和访问统计等功能。

## 游戏目标

平台中存在 **多个** SQL注入漏洞，分布在不同的请求位置和参数类型中。你需要：

1. 使用平台的各项功能
2. 通过抓包工具分析请求参数
3. 找到所有SQL注入漏洞
4. 在漏洞挖掘卡片中提交准确的漏洞信息（URL、参数位置、注入类型）

## 提示

- 漏洞可能存在于各种请求位置：URL参数、表单数据、JSON请求体、Cookie、HTTP头部等
- 不同漏洞的闭合方式不同，需要逐个分析判断
- 部分接口会返回错误信息，部分接口只会返回通用提示
- 熟练使用抓包工具（如Burp Suite、浏览器开发者工具）会很有帮助

## 重置

点击顶部"重置"按钮可重置靶场，清除所有数据和进度。
