# SQL注入基础靶场

## 靶场说明

这是一个SQL注入基础教学靶场，帮助学员理解SQL注入漏洞的原理、类型和基本利用方法。

## 链接信息

- **标题**: SQL注入基础
- **链接**: ./range/inputverify/SQL-Injection/sqlbase/sqlbase/
- **难度**: 基础
- **分类**: 输入验证类漏洞 > SQL注入 > SQL注入基础

## 学习目标

通过本靶场的学习，你将掌握：

1. SQL注入漏洞的产生原理
2. 数字型和字符型注入的区别
3. 如何使用UNION注入获取数据
4. SQL注入的防御方法

## 功能区域

### 区域1：SQL语句调试
输入搜索内容，观察实际执行的SQL语句，体验SQL注入效果。

### 区域2：SQL基础语法说明
学习SELECT、INSERT、UPDATE、DELETE等基本SQL语法。

### 区域3：SQL漏洞原理及利用
理解前后端交互流程、漏洞产生原因、注入类型分类。

### 区域4：SQL注入尝试
3种注入场景实践：数字型、单引号字符型、双引号字符型。

### 区域5：拓展知识讲解
了解堆叠注入、二次注入、宽字节注入等高级技术。

### 区域6：学习完成
标记学习状态完成。

## 使用说明

1. 首次访问时，点击顶部导航栏的"重置数据库"按钮初始化数据库
2. 按顺序展开各个区域学习
3. 在区域1和区域4进行实际操作练习
4. 完成学习后点击"我已掌握"按钮
