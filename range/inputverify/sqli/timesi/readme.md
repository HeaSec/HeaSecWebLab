# 时间盲注靶场

## 简介

天积服务资产查询系统，提供IT资产编号查询功能。系统处于生产模式，不会显示SQL错误信息。正常查询只会返回编号是否存在（已登记/未找到）。

## 学习目标

深入理解MySQL时间盲注的多种延迟技术，掌握通过不同字符串处理函数提取数据库信息的方法，学习绕过空格过滤、关键字过滤和函数名过滤的常见技巧。

## 任务要求

利用时间盲注技术判断当前数据库名的第一位是否为 'h'。需要同时使用延迟技术和字符串处理函数，两个维度各自达标才可解锁星星：

- ⭐ 1星：1种延迟技术 + 1种字符串处理函数
- ⭐⭐ 2星：2种延迟技术 + 3种字符串处理函数
- ⭐⭐⭐ 3星：3种延迟技术 + 5种字符串处理函数

## 操作提示

- 系统处于生产模式，SQL错误信息不会直接显示在页面上
- 页面会显示每次查询的响应时间，关注时间差异
- 系统对输入进行了安全过滤，部分符号、关键字和函数被禁止

## 重置方式

页面右上角重置按钮
