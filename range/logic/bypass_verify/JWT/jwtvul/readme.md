# JWT基础漏洞 靶场

## 靶场介绍

本靶场用于学习 JWT（JSON Web Token）在实际开发中的基础安全问题

## 完成目标

学员需要在每一关中获取管理员相关能力或通关密码，并完成对应验证，最终理解 JWT 设计与验证中的常见风险点。

## 提示信息

- 注意观察登录后的身份信息与系统返回内容。
- JWT 中的内容不一定都是“不可见”的。
- 某些校验一旦缺失，即使 Token 看起来合法，也可能被错误信任。
