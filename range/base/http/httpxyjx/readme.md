# HTTP协议基础 靶场

## 靶场说明

深入理解HTTP协议的工作原理，学习HTTP请求和响应的组成结构、GET和POST请求的区别，以及常见HTTP请求头和响应头的作用。通过实际的HTTP请求操作，观察HTTP协议的实时传输过程。

- **说明**: 本靶场仅介绍HTTP协议的基础知识，如果想要系统学习，推荐免费教程[菜鸟教程](https://www.runoob.com/http/http-tutorial.html)。

## 学习内容

### 1. HTTP请求结构

学习HTTP请求的完整组成，包括请求行（请求方法、URL、HTTP版本）、请求头（各种请求头字段及其作用）和请求体（POST请求的数据内容）。

### 2. HTTP响应结构

掌握HTTP响应的完整组成，包括状态行（HTTP版本、状态码、状态描述）、响应头（各种响应头字段及其作用）和响应体（服务器返回的实际内容）。

### 3. GET与POST请求

理解GET和POST两种常见请求方法的区别、应用场景、数据传输方式以及在实际开发中的选择原则。

### 4. 常见HTTP头字段

了解常见的HTTP请求头（如User-Agent、Accept、Content-Type等）和响应头（如Server、Set-Cookie、Location等）的含义和作用。

### 5. 实时请求观察

通过实际操作，实时观察和分析HTTP请求的完整传输过程，加深对HTTP协议工作原理的理解。
