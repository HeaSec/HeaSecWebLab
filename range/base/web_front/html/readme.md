# HTML语言基础 靶场

## 靶场说明

了解HTML的基本语法、常用标签、属性等。可以通过代码编辑器修改代码理解HTML的工作原理。

- **说明**: 本靶场仅介绍基础的HTML知识，如果想要系统学习，推荐免费教程[W3School](https://www.w3school.com.cn/html/index.asp)或[菜鸟教程](https://www.runoob.com/html/html-tutorial.html)。

## 学习内容

### 1. HTML基本语法

学习HTML元素的基本结构、语法规则、属性使用和注释方法。

### 2. HTML文档结构

了解HTML文档的基础结构，包括文档类型声明、字符编码、页面标题等核心元素。

### 3. 文本标签

掌握HTML中常用的文本相关标签，包括标题、段落、换行、分割线以及各种文本格式化标签。

### 4. 链接和图片

学习超链接和图片标签的各种用法，包括基本链接、图片显示、图片链接等。

### 5. 表格标签

理解表格的完整结构，包括表头、主体、底部，以及表格的基本属性设置。

### 6. 表单基础

掌握表单的基本结构和各种输入类型，包括文本、密码、数字、邮箱、日期、文件上传等。

### 7. CSS基础（拓展）

学习CSS与HTML结合的三种方式（内联样式、内部样式表、外部样式表），以及常用的CSS选择器、盒子模型、定位和布局等样式优化技巧。

 - **说明：** 由于CSS在实际WEB安全中影响较小，因此结合在HTML中进行简单介绍，不做单独的靶场了。
