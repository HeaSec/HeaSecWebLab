# URL任意跳转漏洞靶场

## 靶场介绍

本靶场帮助学习者理解URL任意跳转（Open Redirect）漏洞的原理和常见绕过方式。通过实际操作，掌握如何识别和利用服务端对跳转URL校验不严格的安全隐患。

## 学习目标

- 理解URL任意跳转漏洞的原理
- 掌握常见的绕过白名单检查的方式
- 了解URL跳转漏洞的危害和防御方法

## 任务目标

构造一个URL，使其通过安全检查但实际跳转到 `www.baidu.com`，尝试采用不同的绕过方式获取更多成就。

## 提示

- 系统会检查跳转地址的安全性，不安全的跳转地址会被忽略（后台也不会记录），但这个安全机制可以被绕过。
