# SSRF漏洞 靶场

## 靶场说明

欢迎来到天积识图系统！本系统提供AI图片识别服务，用户输入图片URL，系统将自动获取并识别图片内容。

然而，系统似乎存在某种安全隐患...

## 学习目标

- 理解SSRF（服务端请求伪造）漏洞的原理和危害
- 掌握SSRF漏洞的多种协议利用方式
- 了解内网渗透的基本思路
- 学习如何防御SSRF漏洞

## 任务要求

根据页面中的任务提示，逐步探索并利用系统中的SSRF漏洞，获取秘密字符串完成通关。

## 重置说明

靶场数据可以通过页面右上角的重置按钮进行重置。
重置将清空所有测试数据并恢复初始状态。
