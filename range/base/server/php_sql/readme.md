# 服务端语言基础 靶场

## 靶场说明

了解PHP和SQL的基本语法、表单处理、文件操作、数据库操作以及Cookie和Session管理。可以通过代码示例和预览效果理解服务端脚本的工作原理。

- **说明**: 本靶场仅介绍服务端脚本的基础知识，如果想要系统学习PHP，推荐免费教程[W3School](https://www.w3school.com.cn/php/index.asp)或[菜鸟教程](https://www.runoob.com/php/php-tutorial.html)。

## 学习内容

### 1. PHP基本语法

学习PHP代码的基本结构、在HTML中的嵌入方式、变量和数据类型、控制语句（if-else、for、foreach）以及函数的定义和调用。

### 2. SQL基本语法

掌握SQL语言的语法规则，包括创建表（CREATE TABLE）、查询数据（SELECT）、插入数据（INSERT）、更新数据（UPDATE）和删除数据（DELETE）等常用操作。

### 3. PHP表单处理

理解PHP如何接收和处理HTTP请求，学习超全局变量$_GET和$_POST的使用，以及基本的数据验证和安全处理方法。

### 4. PHP文件处理

学习PHP的文件操作功能，包括文件上传、创建、读取、下载和删除等常用文件处理操作。

### 5. PHP数据库处理

掌握PHP使用PDO（PHP Data Objects）进行数据库操作，包括数据库连接、插入数据、查询数据、更新数据和删除数据的完整流程。

### 6. PHP Cookie和Session处理

了解PHP的Session管理和Cookie使用，学习如何实现用户登录、登出以及"记住我"功能。
